# AI-Assisted Self-Healing Kubernetes Infrastructure

A production-style DevOps platform that detects infrastructure anomalies and automatically heals Kubernetes workloads.

## Architecture

Load Generator → API Service → Redis → Prometheus → Grafana → Self-Healing Controller

## Features

- Microservices deployed on Kubernetes
- Monitoring using Prometheus
- Visualization with Grafana
- Automated anomaly detection
- Self-healing infrastructure automation

## Tech Stack

- Kubernetes
- Docker
- Helm
- Prometheus
- Grafana
- Python
- Terraform
- GitHub Actions

## Project Goal

Simulate a real-world cloud infrastructure platform capable of detecting failures and automatically recovering without human intervention.
